Features

Scroll snapping, intersection-driven active state, progress bar, accessible dot navigation and keyboard control.

Native scroll snap

Uses CSS scroll-snap to lock viewport to full-screen sections for predictable navigation.

Accessible controls

Dot navigation, keyboard shortcuts and semantic markup improve accessibility for keyboard and screen reader users.

Progress & feedback

Top progress bar shows how far you are through the document; sections highlight when in view.
